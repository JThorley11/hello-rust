# Hello Rust Project
This repository is a collection of Rust projects that serve as my first exposure to the Rust programming language, inspired by the Rust handbook.

## Folder Descriptions
**guessing_game/**: This folder contains the first Rust task I completed, the guessing game allowed me to learn about ownership.

**minigrep/**: This folder holds a command line tool that searchs txt files for words, it demonstrates useful patterns like iterators.

**server/**: This folder shows a simple multi-threaded HTTP server with some html interfaces

## How to Run the Projects
Ensure you have Rust installed on your machine. You can install it from [rust-lang.org](https://www.rust-lang.org/).

Navigate to the desired project folder in your terminal.

Run the project using the command:
```pwsh
cargo run
```
These projects were a useful start for me to learn the principles of Rust and I hope to expand on them in the future.
